# 经典的模板

```
<!DOCTYPE html>
<html>
    <head>
        Meta info goes here!
    </head>
    <boddy>
        Content goes here!
    </body>
</html>
```
- DOCTYPE：描述 HTML 的类型。尽管技术上存在不同的类型，但对于你将要编写的 99.999% 的 HTML 来说，<!DOCTYPE html> 是不错的选择。
- <head>：描述有关站点的元信息（例如标题），并提供网页正确渲染和交互，所需的脚本和样式表链接。
- <body>：描述用户将看到的网页上的实际内容。
